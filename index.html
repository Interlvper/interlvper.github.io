PROJECTS

Dynamic Interaction System

UE5 Plugin
The Dynamic Interaction System is an interaction system that allows for communication between actors. It can be used for Players, NPCs, Actors, and much more! The system is written entirely within C++ with all parameters exposed to Blueprints. This includes a detailed debugging system that makes it easy to understand what the system is doing. All it takes to set up the system is an actor component on the actor that will utilize interaction, and an interface implementation on the actor that can be interacted with.
